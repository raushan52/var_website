What is the major Difference between Java and JavaScript

Though Java and JavaScript sound similar and close to each other, there are visible differences in both of them,

Java is an Object Oriented Programming Language whereas JavaScript is an Object Oriented Scripting Language.

Both Java and JavaScript require different plug-ins

While Java created applications which can run on any machine or browser which support Java, JavaScript is a code which runs on a browser only

Codes written in Java have to be compiled while the JavaScript code is all text and no need of compilation.
